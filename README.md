# video-splitter-for-big-size-video-senders
ENGLİSH:
Video splitter for those uploading large videos    

READ BEFORE USİNG:
How it works:
Sender — Split tab:
Select all 3 videos
Set the maximum size (25 MB for the free Discord, 500 MB for Nitro, etc.)
Press "Split & Save" → each video is split into its own folder on the desktop
Receiver — Merge tab:

Download the same application
Select any .part001 file
Press "Merge" → it automatically finds the remaining parts and extracts them as video1.mp4

FOR THE 3.  OPTİON (COMPRESS) YOU NEED TO HAVE FFmpeg İNSTALLED; İF NOT, İNSTALL FROM HERE ffmpeg.org/download.html


TURKİSH:
Büyük videolar yükleyenler için video bölme aracı

KULLANMADAN ÖNCE OKUYUN:
Nasıl çalışır:
Gönderici — Bölme sekmesi:
3 videonun tamamını seçin
Maksimum boyutu ayarlayın (ücretsiz Discord için 25 MB, Nitro için 500 MB, vb.)
"Böl ve Kaydet"e basın → her video masaüstünde kendi klasörüne bölünür
Alıcı — Birleştirme sekmesi:

Aynı uygulamayı indirin
Herhangi bir .part001 dosyasını seçin
"Birleştir"e basın → kalan parçaları otomatik olarak bulur ve video1.mp4 olarak çıkarır
